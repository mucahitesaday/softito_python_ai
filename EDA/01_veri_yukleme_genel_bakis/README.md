# 01 — Veri Yükleme ve Genel Bakış

Bu çalışma CSV verisini pandas ile yükler; ilk/son/rastgele satırları, boyutu,
veri tiplerini, benzersiz değerleri, eksik değerleri ve temel istatistikleri
inceler. Eksik değer haritası ile sütun bazlı eksik değer grafiği oluşturur.

```bash
python EDA/01_veri_yukleme_genel_bakis/veri_yukleme.py
```
