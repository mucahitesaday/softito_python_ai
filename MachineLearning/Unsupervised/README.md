# Unsupervised Learning

Denetimsiz öğrenmede hazır bir hedef sütunu bulunmaz. Amaç, gözlemler arasındaki
benzerlikleri kullanarak doğal grupları, sıra dışı noktaları veya daha düşük
boyutlu temsilleri keşfetmektir.

| No | Çalışma | Kapsam |
|---|---|---|
| 01 | Dijital müşteri segmentasyonu | K-Means, Agglomerative, DBSCAN, GMM, PCA, t-SNE |
| 02 | E-ticaret RFM segmentasyonu | RFM, K seçimi, K-Means, müşteri profilleri |

İki çalışma da veri üretiminden temizlik ve model değerlendirmeye, grafiklerden
CSV/JSON raporlarına kadar uçtan uca çalıştırılabilir.
