# Supervised Learning

Denetimli öğrenmede model, hedef değeri bilinen örneklerden öğrenir. Bu klasörde
hem sayısal değer tahmini (regresyon) hem de sınıf tahmini (sınıflandırma)
problemleri ele alınır.

| No | Algoritma | Problem türü | Projeler |
|---|---|---|---|
| 01 | Linear Regression | Regresyon | Bahşiş ve diyabet ilerleme tahmini |
| 02 | Logistic Regression | Sınıflandırma | Tümör teşhisi ve şarap türü sınıflandırma |
| 03 | Polynomial Regression | Regresyon | Sıcaklık ve enerji talebi |
| 04 | Decision Tree | Sınıflandırma | Çalışan terfi tahmini |
| 05 | KNN ve Naive Bayes | Sınıflandırma | El yazısı rakam tanıma |
| 06 | SVM | Sınıflandırma | Kernel ve karar sınırı analizi |
| 07 | Boosting | Sınıflandırma | Makine arıza riski |

Her bölüm veri hazırlama, eğitim/test ayrımı, ölçeklendirme, model eğitimi,
metrikler, görselleştirme ve otomatik test aşamalarını içerir.
