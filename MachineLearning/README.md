# Machine Learning

Bu klasörde denetimli ve denetimsiz makine öğrenmesi algoritmaları farklı
problemler ve veri setleri üzerinde uygulanır. Her projede veri hazırlama,
eğitim/test ayrımı, model değerlendirme, görselleştirme ve yorumlama adımları
birlikte ele alınır.
